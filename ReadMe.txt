There are totally 5 projects under this solution folder:

1. FootballCore
	Contains the definition of model classes and interfaces.

2. Infrastructure
	Contains the implementation of IFileParser interface. 
	This is low level file structure and content parsing.

3. DomainServices
	Contains business logic utilising the parsed data from file.

4. Football
	FootballApp is to process high level business logic, which is to 
	output information as per requested by the requirements.

5. FootballTest
	Contains two test cases to demonstrate that the basic functionality 
	requirements are met.
